# Zaahir's Personal Website

My little corner of the internet. Check it out [here](https://zaahir.ca).

## Tech Stack

- **SvelteKit** (Svelte 5) - Full-stack framework
- **Tailwind CSS** - Styles
- **Vite** - Build tool
- **Vercel** - Deployment

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Run type checking
npm run check

# Run tests
npm test
```
